# Superchat Log

Superchat/Gift/Membership log viewer for bilibili (Chrome extension)

# 言語・フレームワーク

- JavaScript
- Browserify

# コマンド

```bash
# Initial setup
$ yarn

# Build codes
$ yarn build
```
